프로젝트 개요 (Overview)프로젝트명: K-VIBE-MBTI-Food-Quiz (가칭)목표: 외국인 사용자를 대상으로 한 '나와 가장 닮은 한국 음식'을 찾는 바이럴 웹앱 MVP 제작.기술 스택: HTML, CSS, Vanilla JavaScript (클라이언트 사이드 로직만 사용).핵심 기능: 15개 질문 응답을 기반으로 MBTI-AT 32가지 유형을 분류하고, 각 유형에 매칭된 한국 음식 결과와 'K-Vibe 일치율'을 제공합니다.다국어: 영어 지원 필수 (기타 언어는 텍스트만 준비해 두는 수준).1. HTML (index.html) 구조 정의웹앱은 3개의 주요 페이지/뷰로 구성됩니다. 모든 콘텐츠는 단일 HTML 파일 내에서 JavaScript로 전환됩니다.시작 화면 (Start Screen):제목: "Find Your K-Food Soulmate (Korean Dish MBTI-AT)"사용자 이름 입력 필드 (필수: 바이럴 콘텐츠 개인화)'테스트 시작' 버튼간단한 앱 설명 및 다국어 지원 표시 (예: "Available in English, Japanese, Spanish")테스트 화면 (Quiz Screen):질문 번호 및 진행률 표시 (1/15 및 진행 바)현재 질문 텍스트2개의 선택지 버튼 (MBTI의 이분법적 측정 반영)결과 화면 (Result Screen):사용자 이름 포함된 결과 헤드라인 (예: "[이름] is a Kimchi Jjigae!")결과 음식 이미지 및 이름 (대형)K-Vibe 일치율 (정량적 수치 및 문구)결과 설명 (음식 특징과 MBTI-AT 성향 매칭 설명)공유 버튼 (카카오톡, X/트위터, 링크 복사, 이미지 저장)2. JavaScript (script.js) 로직 및 데이터 정의A. 데이터 구조 정의:questions 배열: 15개 질문 데이터를 저장합니다.각 질문은 5가지 MBTI-AT 축($E/I, S/N, T/F, J/P, A/T$) 중 하나를 측정해야 합니다.질문 객체 구조: id, text_en, options (2개의 선택지), dimension (측정 축: e.g., 'E', 'S', 'T', 'J', 'A')results 객체/맵: 32가지 유형(E_S_T_J_A, E_S_T_J_T 등)을 키로, 매칭되는 한국 음식 데이터를 값으로 저장합니다.결과 객체 구조: food_name_en, image_url, description_en, compatibility_food (궁합 음식), mbti_at_vibeB. 핵심 로직 구현 단계:변수 초기화: 사용자 이름, 현재 질문 번호, 각 5개 축의 점수 카운터를 저장할 변수 초기화.startQuiz() 함수: 시작 버튼 클릭 시 호출되며, 이름 저장 및 테스트 화면으로 전환.loadQuestion() 함수: questions 배열에서 다음 질문을 불러와 화면에 표시.handleAnswer(choice_dimension) 함수:선택된 선택지에 따라 해당 dimension의 점수 카운터 증가.다음 질문으로 넘어가기 (loadQuestion()) 또는 테스트 완료 시 showResult() 호출.showResult() 함수 (가장 중요):MBTI-AT 유형 결정: 5개 축 각각의 카운터 점수를 비교하여 더 높은 쪽(예: E vs I)을 최종 유형으로 결정합니다. (총 32가지 조합)K-Vibe 일치율 계산: MBTI-AT 유형 결과(예: I, F, J, A)를 기반으로 'K-Vibe 일치율 계산 로직' (상단 분석 참고)을 적용하여 $0\% \sim 100\%$ 사이의 수치를 도출.결과 매칭 및 표시: 최종 32가지 유형 키를 사용하여 results 맵에서 해당 음식 정보를 불러와 결과 화면에 모든 정보를 표시.3. CSS (style.css) 스타일링 요구사항반응형 디자인: 모바일 환경(스마트폰 화면)에 최적화된 심플하고 깔끔한 디자인 필수.한국적 요소: K-Food나 K-Culture의 느낌을 살린 색상 팔레트와 폰트를 활용.바이럴 강조: 결과 음식 이미지와 'K-Vibe 일치율' 숫자가 눈에 띄도록 크고 매력적으로 디자인.개발 작업 요청 (프롬프트 최종 정리)"Replit 환경에서 동작하는 'MBTI-AT 기반 한국 음식 퀴즈' 웹앱 MVP를 구축해 주세요. HTML, CSS, JavaScript 파일을 생성하고, 다국어 (영어)를 기본으로 사용하며, 아래 사항을 구현합니다.index.html: 시작, 테스트, 결과 3단계 뷰를 포함하는 구조. 사용자 이름 입력 및 '테스트 시작' 버튼 포함.script.js:questions 데이터 (15개 질문, 5개 축 측정 로직) 정의.results 데이터 (32가지 유형별 한국 음식 매칭) 정의.showResult() 함수 내에 MBTI-AT 최종 분류 로직 및 K-Vibe 일치율 (0~100%) 계산 로직을 구현.style.css: 모바일 최적화 및 바이럴 공유에 적합한 시각적 디자인 적용."
